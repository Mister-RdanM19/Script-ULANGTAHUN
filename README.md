# 🌸 Project HBD n3i - script ulang tahun

Sistem antarmuka web khusus yang dirancang untuk merayakan hari spesial bagi **n3i**. Proyek ini merupakan kompilasi dari 30 modul ucapan interaktif yang dikemas dalam satu dashboard bernuansa romantis dan estetis.

---

## Main Features

* **Cyber-Romance Dashboard**: Antarmuka utama (`index.html`) yang menggunakan desain *glassmorphism* dengan palet warna lembut.
* **Dynamic Particles**: Efek kelopak bunga dan hati yang jatuh secara real-time untuk menciptakan suasana hangat.
* **30 Unique Modules**: Terdiri dari 30 file berbeda (1.html - 30.html) dengan berbagai gaya, mulai dari tampilan terminal hingga desain *starry night*.
* **Mobile Responsive**: Dioptimalkan untuk tampil sempurna di perangkat *mobile* maupun *desktop*.
* **Clean Architecture**: Struktur kode yang rapi menggunakan standar teknologi web modern (HTML5, CSS3, dan JavaScript).
